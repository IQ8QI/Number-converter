# Roman to decimal
Simple CLI tool to convert number between Roman and decimal number<br>
Accepts only positive values in range <1-3999><br>

# Usage
Convert Roman number to decimal

    romtodec MMMCV
    romtodec --roman CCVI
    romtodec --ro XXVIII
    romtodec -r CCLXXVI

Convert decimal number to Roman

    romtodec --decimal 3420
    romtodec --dec 123
    romtodec -d 14

Display help

    romtodec
    romtodec ?
    romtodec --help
    romtodec -h
